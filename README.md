# Reproductor-WEB-de-IPTV
Es una pagina web donde ingresas lista M3U / M3U8 o URL tipo &amp;type=m3u_plus, podras guardar la lista con el nombre que quieras, tambien si tu iptv te lo permite, podras ver 4 canales a la vez en MultiTV. Facil de utilizar tarda algunas veces depende de la cantidad de canales que tienes.
# IPTV PRO MAX

## Características
- Reproducción de listas M3U/M3U o URL tipo &amp;type=m3u_plus8 
- Interfaz moderna 
- Búsqueda y filtrado de canales
- Categorización automática
- Soporte para logos de canales
- Tema oscuro con gradiente
- Sin publicidad
- Persistencia de la última lista utilizada

## Uso
1. Abre la pagina
2. Ingresa la URL de tu lista M3U/M3U8 o URL tipo &amp;type=m3u_plus
3. Navega por los canales disponibles
4. Utiliza la barra de búsqueda para encontrar canales específicos
5. Filtra por categorías usando los chips
6. Toca en un canal para reproducirlo
